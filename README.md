## What I've built

**[homelab-monitoring](https://github.com/sh3r44z/homelab-monitoring)** — A production-inspired observability stack running on my home server. Prometheus scrapes Node Exporter and cAdvisor, Grafana visualises it, Alertmanager routes alerts to n8n which handles the notification logic. Because hardcoding Slack webhooks into config files is not the move.

**[trail-runner-mailer]— Automated pipeline that scrapes trail running events from a JS-rendered site using Playwright, packages them, and fires off a digest email on a monthly GitHub Actions cron. Terraform provisions the AWS infrastructure. Prometheus Pushgateway tracks run metrics.

**[learn app](https://www.kadermaybe.com)** — A personal study tool I built and deployed for tracking my DevOps learning. Because Notion wasn't cutting it and building your own tools is more fun anyway.
